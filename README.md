## 下载

### Docker

```
docker pull apachecn0/as-dev-essense-zh
docker run -tid -p <port>:80 apachecn0/as-dev-essense-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g as-dev-essense-zh
as-dev-essense-zh <port>
# 访问 http://localhost:{port} 查看文档
```
